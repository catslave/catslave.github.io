# JDK 7

## 泛型类型推断

**什么是类型推断**

JDK 7 以前的泛型使用：

{% highlight java %}

Map<String, String> myMap = new HashMap<String, String>();

{% endhighlight %}

在变量声明和赋值时，两侧都要加上泛型类型。

JDK 7 以后：

{% highlight java %}

Map<String, String> myMap = new HashMap<>();

{% endhighlight %}

编译器会根据变量声明时的泛型类型自动推断出实例化时的泛型类型。
